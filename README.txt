1) Eclipse Project (DrawTouchDrive_00001.rar) with an Android 
application that shows how to customize a view and allow the user 
to touch drive simple objects (circles and crosses) via canvas 
redrawing.

2) Targets: Android 2.3+, Eclipse Helios 3.6

3) To start: 
   a) unzip into a folder 
   b) open in Eclipse 
   c) Run as Android application
